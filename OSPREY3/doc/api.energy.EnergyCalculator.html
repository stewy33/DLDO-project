EnergyCalculator¶

Java class: edu.duke.cs.osprey.energy.EnergyCalculator

Computes the energy of a molecule fragment from a conformation space using the desired forcefield parameters and residue interactions.

Residue interactions are specified via ResidueInteractions instances. Forcefield implementations are chosen via the type argument. If no type is specified, the best forcefield available implementation is automatically chosen based on the parallelism argument.

If a fragment has continuous degrees of freedom, minimization will be performed before forcefield evaluation.
