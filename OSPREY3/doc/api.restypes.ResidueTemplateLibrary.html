ResidueTemplateLibrary¶

Java class: edu.duke.cs.osprey.restypes.ResidueTemplateLibrary

Library of ResidueTemplate instances used by Strand instances to define discrete flexibilty.

See Python script examples/python.GMEC/templateLibrary.py in your Osprey distribution for example usage.
