ConfEnergyCalculator¶

Java class: edu.duke.cs.osprey.energy.ConfEnergyCalculator

Calculate energy for molecules created from conformation spaces.

Provides support for applying conformation energy modifications, such as reference energies, residue entropies, and energy partitions.
