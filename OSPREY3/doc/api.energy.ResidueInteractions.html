ResidueInteractions¶

Java class: edu.duke.cs.osprey.energy.ResidueInteractions

Represents interactions between residues (or within single residues) based on residue numbers.

Does not rely on specific molecules, so residue interactions are portable between different molecules, or molecule copies.

The ResInterGen helper class can assist with creating residue interactions.
